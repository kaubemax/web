Static frontend, Supabase backend

Built for GitHub Pages.

The frontend stays static and is served from GitHub Pages. Published entries, photos, and section visibility come live from Supabase with public read-only access, while every write is protected by Eva's account and row-level security. Data is structured from the start — ingredients, method steps, and espresso extraction values are stored as queryable rows for later analysis.
